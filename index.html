Языки разметки

Сравнительный обзор XTL, HTML, YAME и YTL

HTML

HyperText Markup Language

Стандартный язык разметки для создания веб-страниц и веб-приложений.

<!-- Пример HTML документа -->
<!DOCTYPE html>
<html lang="ru">
<head>
  <title>Моя страница</title>
</head>
<body>
  <h1>Привет, мир!</h1>
</body>
</html>

XTL

eXtensible Template Language

Язык шаблонов на основе XML для генерации различных форматов вывода.

<!-- Пример XTL шаблона -->
<template name="userCard">
  <div class="user">
    <h2>{{name}}</h2>
    <p>Email: {{email}}</p>
    <if condition="admin">
      <span>Администратор</span>
    </if>
  </div>
</template>

YAME

Yet Another Markup Extension

Расширяемый язык разметки с поддержкой макросов и пользовательских тегов.

# Пример YAME документа
@macro button(text, color="blue") {
  <button class="btn btn-{{color}}">
    {{text}}
  </button>
}

@import "components/header.yame"

<page title="Главная">
  <header>
    <h1>Добро пожаловать!</h1>
  </header>
  @button("Начать", "green")
</page>

YTL

YAML Template Language

Шаблонный язык на основе YAML с лаконичным синтаксисом для генерации контента.

# Пример YTL шаблона
template: "productList"
data:
  - name: "Ноутбук"
    price: "$999"
    inStock: true
  - name: "Мышь"
    price: "$25"
    inStock: false
output:
  html:
    - tag: "ul"
      class: "products"
      children:
        for: "product in data"
        tag: "li"
        text: "{{product.name}} - {{product.price}}"
